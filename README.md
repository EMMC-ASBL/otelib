# OntoTransServer


## Running the compevo case !
Prerequisites: jupyter lab is installed and oteapi is running!

In jupyter lab open the excecute-compevo.ipynb and update the fields <OTEAPI_ENDPOINT>  and <GITLAB_ACCESS_TOKEN> 

GITLAB_ACCESS_TOKEN can be generated from  https://gitlab.cc-asp.fraunhofer.de/-/profile/personal_access_tokens  by providing the scopes:api, read user, read_repo and read_api
